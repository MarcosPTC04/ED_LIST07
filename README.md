## 📌 Descrição

Este repositório apresenta a implementação de uma Lista Sequencial (TAD) em Java utilizando arrays. O objetivo é praticar conceitos fundamentais de estrutura de dados, como manipulação de índices, controle de capacidade e lógica de deslocamento (shift) para inserção e remoção de elementos.

O projeto demonstra como os dados são organizados sequencialmente na memória e como as operações funcionam internamente, especialmente em casos que exigem movimentação de elementos dentro do array.

📋 Estrutura do Código

O projeto está organizado nas seguintes partes:

---

## 🔹 Classe MyArrayList

Responsável por implementar a lista sequencial, contendo métodos de:

Inserção (addFirst, addLast, insertAt, addSorted)
Remoção (removeFirst, removeLast, removeAt, remove)
Busca (find, get)
Atualização (set)
Verificação (isEmpty, isFull, count)
🔹 Método display()

Utilizado para exibir os elementos armazenados na lista.

---

## 🔹 Classe Main

Responsável por testar todas as operações implementadas e validar o funcionamento da estrutura.

⚙️ Observações sobre a implementação
A lista é baseada em um array de tamanho fixo.
As operações utilizam lógica de deslocamento (shift) dos elementos.
O controle é feito pela variável count.
Possui tratamento de erros (lista cheia, vazia e índice inválido).
